## Alex Finn's OpenClaw Setup Advice (Moonshot Podcast, March 2026)

**Installation & Setup:**
- OpenClaw is free & open source
- Run LOCAL (not VPS) - easier, secure by default, better integration
- Don't rush to buy hardware - start with what you have, upgrade later
- Mac Mini is best value ($600) if you need to buy

**Model Options:**
- Anthropic Opus 4.6 = best brain, ~$200/month (risky - can get banned)
- OpenAI = warmer than Miniax, good if you already have subscription
- Miniax = cheapest option, still powerful enough

**Messaging:** Telegram is best (recommended), Discord for advanced workflows

**First Things to Do:**
1. **Brain dump** - Tell it everything about you (background, preferences, goals, ambitions)
2. **Morning brief** - Schedule a daily automated briefing (weather, news, tasks, recommendations)
3. **Mission control** - Build a custom dashboard for tools/workflows

**Advanced Concepts:**
- **Brain & Muscles**: Use Opus as the brain, cheaper models (Codex, XAI, Brave) as muscles for specific tasks
- **Local models**: The endgame - run locally for unlimited tokens + privacy
- **Reverse prompting**: Ask it questions instead of giving instructions - "Based on what you know about me, what should we build?"

**Security Rules:**
- It has admin access to EVERYTHING on your computer
- DON'T expose it to the world (no group chats, no public access)
- Think before every prompt - could it be prompt injected?
- If you don't want it accessing something, don't be logged into it on that computer
