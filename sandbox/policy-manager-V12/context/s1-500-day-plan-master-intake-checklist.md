# S1 500 Day Plan, Master Intake Checklist

Use this checklist to gather the information needed to build and run the S1 summit under the 500 Day Plan.

## 1. Shared financial foundation
- monthly P&L by clinic
- consolidated CSI monthly P&L
- balance sheet
- debt summary
- ownership / payout structure
- current reporting package
- any existing dashboards
- known governance constraints

## 2. COGS Discipline
- clinic-level COGS by month
- supplier list
- purchase volumes by category
- supplier price changes
- inventory procedures by clinic
- expiry / wastage / shrinkage issues
- markup logic on products and supplies
- current purchasing controls

## 3. Labour Alignment
- vet payroll by clinic
- non-vet payroll by clinic
- staffing roster by role
- schedules / hours worked
- production by doctor
- appointment volume by clinic
- overtime patterns
- overstaffing / understaffing concerns

## 4. PAT / Pricing Improvement
- PMS price lists for each clinic
- date of last price update by code
- usage volume by service
- PAT / average transaction data
- missed charge examples
- pricing reports already created
- shoppable service list
- benchmark / competitor pricing

## 5. Operational Expansion
- current clinic hours
- room / doctor / staffing capacity
- UPC utilization data
- evening / weekend utilization
- turnaway or overflow demand
- space / equipment available
- current operational bottlenecks

## 6. Variable Cost Review
- recurring monthly expense list
- subscriptions / software list
- vendor contracts
- discretionary expense list
- service agreements
- unusual overhead growth
- questionable or legacy expenses

## 7. Interest / Cash Flow Improvement
- full debt list
- lenders
- balances
- rates
- payment terms
- monthly cash obligations
- shareholder loan details
- tax debt details
- available cash sources
- major inflow / outflow timing

## 8. Build priorities
- which lane should start first
- which lane already has usable data
- which lane is most urgent
- which lane has the fastest financial payoff
- which lane is blocked by missing information

## 9. Scott input / judgment layer
- what Scott already knows is broken
- where Scott thinks money is being lost fastest
- which leaders / managers are strong
- which people / roles are blockers
- what should not be touched yet
- what is politically sensitive

## 10. Success definition
- what counts as a win in the first 37 days
- what counts as a win in the first 100 days
- what must be true at 500 days
