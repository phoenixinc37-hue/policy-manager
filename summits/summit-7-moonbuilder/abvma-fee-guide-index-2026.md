## Usage Notes for Clinic IQ Reports
- **Canine/Feline only clinics:** Use Core + Common categories
- **General practice with exotics:** Add Exotics section
- **Emergency/after-hours clinics:** Ensure after-hours exam and emergency surcharge codes are included
- **Specialist clinics:** AbVMA notes specialists typically charge 15–20% above guide
- **All prices are suggested only** — not binding, used as benchmark comparison only
- **Annual update required** — new fee guide issued each year
