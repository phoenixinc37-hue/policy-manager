# TASK 3: VIRTUAL DELIVERY PLAN
## "Advice to a Teammate" — Remote Delivery

---

## FORMAT OPTIONS

### Option A: Zoom/Google Meet (Recommended)
- Presenter shares screen with workbook
- Athletes follow along on their own paper workbook
- breakout rooms for small group discussions
- Presenter's camera on to build connection

### Option B: Hybrid
- Some in-person, some remote
- Good for spreading across locations

---

## STRUCTURE (45-60 min)

### Pre-Call
- Send workbook PDF to team
- Ask athletes to print or have ready
- Send Zoom link

### Call Flow (Zoom)

| Time | Content |
|------|---------|
| 0-5 min | Intro + "Why this matters" |
| 5-10 min | Page 1-2: The Moment + Teammate identity |
| 10-20 min | Page 3-5: What supportive teammate looks like |
| 20-30 min | Page 6-7: Being that teammate to yourself + scars |
| 30-40 min | Page 8: Practice exercise (write) |
| 40-50 min | Page 9: Reset Process |
| 50-60 min | Page 10: The Scar ceremony + coin |

### Post-Call
- Send recording link
- Send digital copy of coin image (for printing)
- Follow up in 1 week

---

## PRESENTATION TIPS (FOR IZZY)

- NOT a lecture — it's a conversation
- Ask questions, get them talking
- Use breakout rooms for discussions
- Share screen, walk through workbook pages
- Have athletes write in workbook during exercises

# TASK 4: PHONE DELIVERY PLAN
## Digital Workbook + Phone-First Experience

---

## CONCEPT

Instead of paper workbook, athletes follow on their phones:
- Workbook becomes interactive digital experience
- No printing needed
- Can be done individually or in group
- Presenter guides via Zoom + phone combo

---

## OPTIONS

### Option A: Website/App (Future)
- Build Achilles Athletics web app
- Workbook inside app
- Progress tracked
- Coin as digital badge

### Option B: PDF + Interactive (Now)
- Send PDF workbook
- Athletes annotate on phone (GoodNotes, PDF Expert)
- Exercises done in notes app

### Option C: Notion/Google Doc (Simple)
- Create Google Doc workbook
- Share with athletes
- They type responses
- Real-time editing

---

## RECOMMENDED: GOOGLE DOC APPROACH

**Setup:**
1. Create "Advice to a Teammate" Google Doc
2. Share with "view only" to team
3. Athletes make copy
4. They type answers in their copy

**Benefits:**
- Free
- Everyone has phone access
- Easy to share
- Presenter can see if they've done prep

---

## SESSION FLOW (Phone + Zoom)

| Phase | Delivery |
|-------|----------|
| Intro | Zoom (presenter on video) |
| Content | Zoom + workbook on phone |
| Exercises | Athletes write in Google Doc on phone |
| Discussion | Zoom breakout rooms |
| Ceremony | Zoom — presenter shows coin |
